Projet Git_TP_2
Mon projet est de personnaliser un template Bootstrap qui affiche une page web avec un caroussel et une barre de menu responsive.

1ère étape :
c'est que le caroussel qui pour l'instant n'a pas d'image affiche des image "Placeholder" tiré du site web http://lorempixel.com. 
Pour cela je vais modifier le fichier index.html et remplacer l'attribut sources de la balise img par 
src="http://lorempixel.com/1200/500/people pour la ligne 84 
src="http://lorempixel.com/1200/500/sport pour la ligne 94
src="http://lorempixel.com/1200/500/nature pour la ligne 104

2ème étape : 

Créer un fichier CSS : styles.css
Modifier la barre de navigation pour que la couleur du lien actif devient bleu
